foreach的基础知识

一个foreach使用的官方小例子

foreach包含以下的一些属性：

  • collection: 必填项，值为要迭代循环的属性名。分多种情况
  • item: 变量名，值为从迭代对象中取出的每一个值。
  • index: 索引的属性名，在集合数组情况下值为当前索引值，当迭代循环的对象是Map类型时，这个值为Map的key(键值)
  • open: 整个循环内容开头的字符串
  • close: 整个循环内容结尾的字符串
  • separator: 每次循环的分隔符

collection的默认处理逻辑为DefaultSqlSession的方法

可以看出默认是由三种类型

  • collection
  • list
  • array

除此之外可以通过@param来指定类型别名

foreach可能会面对的循环情况

  • 只有一个数组参数或集合参数(这里简单说明一下对应关系)
    • List: “list”/@param
    • Map: key的类型/@param/_parameter
    • []数组: “array”/@param
  • 有多个参数
    • 建议使用@param
  • 参数是Map类型
    • 建议使用@param
    • 默认值_parameter
    • key的类型
  • 参数是一个对象
    • 建议使用属性名
    • @param
